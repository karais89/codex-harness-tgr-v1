# 마일스톤 체크리스트 템플릿

## 목적

이 문서는 각 마일스톤의 완료 여부를 작은 체크리스트 단위로 판정하기 위한 공통 형식입니다.

각 마일스톤 체크리스트는 `docs/project-goal.md`에 적힌 완료 기준을 더 작고 확인 가능한 항목으로 나눕니다. 체크리스트만 보아도 해당 마일스톤이 완료되었는지 판단할 수 있어야 합니다.

## 완료 판정 규칙

- 필수 체크리스트의 모든 항목이 완료되어야 합니다.
- 각 완료 항목에는 확인 방법과 검증 증거가 남아야 합니다.
- 검증하지 못한 항목이 있으면 마일스톤을 완료로 판정하지 않습니다.
- 예외가 필요한 경우 `미확인/예외`에 이유와 영향을 적고, 최종 판정에서 완료 여부를 명확히 기록합니다.

## 최종 판정

- 다음 조치: 남은 항목이 있으면 적습니다.
