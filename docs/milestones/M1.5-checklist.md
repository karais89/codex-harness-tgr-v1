# M1.5 체크리스트 - Tiny Garden 기획/아트 기준 정리

## 목적

이 문서는 `docs/project-goal.md`의 `M1.5. Tiny Garden 기획/아트 기준 정리` 완료 기준을 작은 확인 항목으로 나누어, M1.5 완료 여부를 체크리스트만으로 판정하기 위해 작성합니다.

M1.5의 목표는 M2 Scene 구성을 시작하기 전에 Tiny Garden Restore의 최소 기획, 아트 방향, 이미지 생성 기준을 고정하는 것입니다.

## 완료 판정 규칙

- 필수 체크리스트의 모든 항목이 완료되어야 M1.5를 완료로 판정합니다.
- 각 항목은 확인한 문서 경로와 완료 증거를 함께 남깁니다.
- 기획 문서, 아트 컨셉 문서, 이미지 프롬프트 문서, 컨셉 이미지, 질문/답변 기반 결정 기록 중 하나라도 빠지면 M1.5를 완료로 판정하지 않습니다.
- M1.5는 문서와 컨셉 참고 이미지까지만 다루며, Unity Scene이나 `Assets/` 변경은 M2 이후 항목으로 남깁니다.

## 필수 체크리스트

- [x] M1.5-01: 기획 문서가 존재합니다.
  - 확인 방법: `docs/creative/tiny-garden-creative-spec.md`가 있는지 확인합니다.
  - 완료 증거: 문서에 플레이 감각, 복구 대상, 상호작용, M2~M5 playable loop 기준, 하지 않을 것이 기록되어 있습니다.
- [x] M1.5-02: 아트 컨셉 문서가 존재합니다.
  - 확인 방법: `docs/art/tiny-garden-art-concept.md`가 있는지 확인합니다.
  - 완료 증거: 문서에 아늑한 2D 스타일, 봄 정원 팔레트, top-down 시점, 작은 정원사, 방치/복구 대비 기준이 기록되어 있습니다.
- [x] M1.5-03: 이미지 생성 프롬프트 문서가 존재합니다.
  - 확인 방법: `docs/art/tiny-garden-image-prompts.md`가 있는지 확인합니다.
  - 완료 증거: 문서에 최종 프롬프트, 피해야 할 요소, 생성 결과 검수 기준이 기록되어 있습니다.
- [x] M1.5-04: 실제 컨셉 이미지가 존재합니다.
  - 확인 방법: `docs/art/images/tiny-garden-concept-board-01.png`가 있는지 확인합니다.
  - 완료 증거: 컨셉 참고용 PNG 이미지가 프로젝트 문서 경로에 저장되어 있습니다.
- [x] M1.5-05: 질문/답변 기반 결정 기록이 존재합니다.
  - 확인 방법: `docs/creative/tiny-garden-creative-spec.md`의 `결정 기록` 항목을 확인합니다.
  - 완료 증거: M1.5 필수 게이트, 질문 방식, 컨셉 이미지 용도, 플레이 감각, 복구 대상, 상호작용, 아트 스타일, 색감, Player, 시점, 방치/복구 대비가 기록되어 있습니다.
- [x] M1.5-06: M2에서 참조할 기준이 명확합니다.
  - 확인 방법: 기획 문서와 아트 문서를 함께 확인해 M2 Scene 구성 기준이 충돌하지 않는지 확인합니다.
  - 완료 증거: M2 기준은 작은 top-down 정원, 작은 정원사 Player, 잡초/시든 꽃/더러운 타일 복구 대상 배치 공간, 방치/복구 대비 표현으로 정리되어 있습니다.
