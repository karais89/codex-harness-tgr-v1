# V2-M1 체크리스트 - AI 실수 장부 생성

## 목적

이 문서는 `docs/project-goal-v2.md`의 `V2-M1. AI 실수 장부 생성` 완료 기준을 작은 확인 항목으로 나누어, v2 시작 시점에 `docs/ai-mistakes.md`를 경량 재발 방지 장부로 만들기 위해 작성합니다.

## 완료 판정 규칙

- `docs/ai-mistakes.md`가 존재해야 합니다.
- 장부는 memory-bank가 아니라 반복 실수 방지용이어야 합니다.
- 첫 기록은 이번 v2 범위 축소/누락 사례를 포함해야 합니다.
- 단순 취향이나 추측을 기록 대상으로 삼으면 완료로 판정하지 않습니다.

## 필수 체크리스트

- [ ] V2-M1-01: `docs/ai-mistakes.md`를 생성합니다.
  - 확인 방법: 파일 존재 여부를 확인합니다.
  - 완료 증거: 파일 경로를 기록합니다.
- [ ] V2-M1-02: 문서 목적을 명확히 적습니다.
  - 확인 방법: memory-bank가 아니라 경량 재발 방지 장부라고 설명되어 있는지 확인합니다.
  - 완료 증거: 목적 섹션의 핵심 문장을 기록합니다.
- [ ] V2-M1-03: 기록 대상과 기록하지 않을 대상을 구분합니다.
  - 확인 방법: 반복 가능성이 있는 구체적 실수와 기록 제외 항목이 분리되어 있는지 확인합니다.
  - 완료 증거: 기준 요약을 기록합니다.
- [ ] V2-M1-04: 기록 형식을 고정합니다.
  - 확인 방법: `실수`, `영향`, `재발 방지 규칙`, `관련 근거` 형식이 있는지 확인합니다.
  - 완료 증거: 기록 템플릿을 기록합니다.
- [ ] V2-M1-05: 첫 실수 사례를 기록합니다.
  - 확인 방법: v2 범위를 사용자 의도보다 좁게 해석해 이미지 교체와 AI 실수 장부를 누락한 사례가 있는지 확인합니다.
  - 완료 증거: 첫 기록 제목과 재발 방지 규칙을 기록합니다.
