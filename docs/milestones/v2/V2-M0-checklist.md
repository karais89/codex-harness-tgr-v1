# V2-M0 체크리스트 - v2 범위 재확정과 현재 상태 확인

## 목적

이 문서는 `docs/project-goal-v2.md`의 `V2-M0. v2 범위 재확정과 현재 상태 확인` 완료 기준을 작은 확인 항목으로 나누어, v2 착수 전 범위와 현재 repo 상태를 다시 고정하기 위해 작성합니다.

V2-M0의 목표는 v2의 세 축인 이미지 교체, AI 실수 장부, UX/검증 보강이 모두 기준 문서에 반영되어 있는지 확인하는 것입니다.

## 완료 판정 규칙

- `docs/project-goal-v2.md`가 사용자와 확정한 v2 범위를 반영해야 합니다.
- 현재 작업 트리와 `.codex` hook/skill 상태를 확인해야 합니다.
- 기존 v1 마일스톤 경로와 v2 체크리스트 경로가 구분되어야 합니다.
- 확인하지 못한 항목이 있으면 V2-M0를 완료로 판정하지 않습니다.
