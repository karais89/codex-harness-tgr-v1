# Concept 후보 정리

상태: 진행 전

## 목적

이 문서는 `01-brainstorming.md`의 아이디어 보드에서 관심이 생긴 후보를 조금 더 정리하는 공간이다.

여기서도 후보는 확정 사양이 아니다. 선택, 보류, 폐기 이유를 더 분명히 남기는 것이 목적이며, gameplay ExecPlan이나 Unity 구현 기준으로 사용하려면 별도 `docs/design` 승인 절차가 필요하다.

## 사용 조건

다음 중 하나일 때 이 문서를 갱신한다.

- 아이디어 보드의 후보가 `관심 있음` 상태가 되었다.
- 여러 후보 중 하나를 `선택`, `보류`, `폐기`로 정리해야 한다.
- 후보의 매력과 우려를 보드 카드보다 조금 더 길게 남길 필요가 있다.

초기에는 점수표나 우선순위 평가표를 두지 않는다. 후보가 충분히 쌓인 뒤 비교 기준이 필요해지면 별도로 논의한다.

## 폐기된 후보

### C-01. 작은 분수 복구 노드 퍼즐

상태: 폐기

요약:

```text
작은 노드 맵에서
제한된 행동 안에
접근 경로를 열어가며
작은 분수를 복구하는
따뜻한 정원 복구 퍼즐
```

폐기 이유:

- 사용자가 원한 가벼운 아이디어 보드보다 첫 playable의 세부 gameplay design 사양에 가깝다.
- 행동 횟수, 노드 수, 상태명, UI 조작, 성공/실패 문구가 creative 단계에서 확정된 것처럼 기록되었다.
- 이 후보를 유지하면 이후 design 승인 없이 구현 기준처럼 재사용될 위험이 있다.

보존 위치:

- 당시 Q&A 원문: `04-brainstorming-qna-2026-05-19.md`
- 폐기 요약 카드: `01-brainstorming.md`

현재 채택 후보가 아니며, `docs/design` 승인 없이 gameplay ExecPlan이나 Unity 구현 기준으로 사용할 수 없다.
