# 생성 이미지 결과

상태: 작성 전

## 목적

이 문서는 생성 이미지 결과와 채택/보류/폐기 이유를 기록하기 위한 공간이다.

## 결과 목록

아직 없음.

## 기록 형식

```md
### 이미지 제목

- 파일: `docs/art/images/<filename>`
- 프롬프트: `02-image-prompts.md`의 항목 참조 또는 직접 기록
- 상태: 채택 / 보류 / 폐기
- 이유:
- Unity 에셋 사용 여부: 미승인
```

## 주의

`docs/art/images/`에 이미지가 저장되어 있어도 실제 Unity 에셋 사용이 승인된 것은 아니다. Unity 에셋으로 사용하려면 별도 design 또는 ExecPlan 승인 절차를 거친다.
